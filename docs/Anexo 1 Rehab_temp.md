| **4. DESCRIPCIÓN Y RAZÓN DE SER DEL PROYECTO.** Justificación y       |
| pertinencia                                                           |
+=======================================================================+
| Smart Rehab es un proyecto innovador centrado en el desarrollo de un  |
| sistema inteligente que se acopla a las barras de sujeción utilizadas |
| en procesos de rehabilitación física por personas con dificultades de |
| movilidad. El dispositivo incorpora células de carga, que permiten    |
| registrar de manera precisa el tiempo de uso y la fuerza de apoyo     |
| ejercida sobre las barras durante los ejercicios realizados con el    |
| fisioterapeuta.                                                       |
|                                                                       |
| El sistema recogerá estos datos y los visualizará de forma clara e    |
| intuitiva para profesionales de la salud, facilitando un seguimiento  |
| objetivo de la evolución del paciente. Además, integrará inteligencia |
| artificial para analizar grandes volúmenes de datos y generar un      |
| diagnóstico orientativo, que complemente el criterio clínico del      |
| fisioterapeuta y permita personalizar el tratamiento rehabilitador.   |
|                                                                       |
| La solución se instalará sobre barras existentes o rediseñadas, y se  |
| prevé su despliegue en centros sanitarios y educativos, como          |
| herramienta de trabajo clínico y también como recurso didáctico para  |
| el alumnado de FP.                                                    |
|                                                                       |
| **Justificación y Pertinencia**                                       |
|                                                                       |
| La rehabilitación física es un proceso complejo y altamente           |
| individualizado. Hoy en día, muchas decisiones clínicas se basan      |
| exclusivamente en la observación directa del fisioterapeuta, sin      |
| apoyo cuantitativo. Smart Rehab Bar nace para cubrir ese vacío,       |
| ofreciendo datos objetivos que mejoren el seguimiento del paciente y  |
| aumenten la eficacia de la intervención.                              |
|                                                                       |
| El proyecto aporta valor a nivel social y educativo: mejora el        |
| cuidado de personas en rehabilitación, introduce tecnologías          |
| emergentes en el ámbito sociosanitario y ofrece un campo de           |
| aplicación real para el alumnado de Formación Profesional,            |
| especialmente en las ramas de electrónica y sanidad.                  |
|                                                                       |
| El proyecto Smart Rehab Bar se alinea directamente con el nodo de     |
| Salud y Cuidados, una de las áreas prioritarias del ecosistema de     |
| innovación de FP de Euskadi, al centrarse en soluciones tecnológicas  |
| para mejorar la calidad de vida de las personas.                      |
|                                                                       |
| Las líneas de trabajo de Smart Rehab Bar se encuentran alineadas con  |
| varias prioridades estratégicas definidas en el marco del RIS3        |
| Euskadi 2030, especialmente en aquellas relacionadas con la salud y   |
| el desarrollo tecnológico orientado a las personas.                   |
|                                                                       |
| En primer lugar, el proyecto se inscribe claramente en la línea de    |
| Salud Personalizada y Activa, ya que su objetivo principal es mejorar |
| los procesos de rehabilitación a través de la recogida de datos       |
| individualizados y su análisis mediante inteligencia artificial. Esta |
| personalización del seguimiento terapéutico permite adaptar los       |
| tratamientos a las necesidades concretas de cada persona,             |
| contribuyendo a una atención sanitaria más eficaz, segura y centrada  |
| en el paciente.                                                       |
|                                                                       |
| Por otro lado, Smart Rehab Bar también se alinea con la prioridad de  |
| Tecnologías Facilitadoras Esenciales, al incorporar sensórica         |
| avanzada e inteligencia artificial para interpretar los datos         |
| recogidos durante las sesiones de rehabilitación. Este enfoque        |
| tecnológico no solo optimiza el proceso clínico, sino que también     |
| representa un avance significativo en la aplicación de soluciones     |
| digitales al sector sociosanitario.                                   |
|                                                                       |
| El proyecto se alinea con el Eje A del VI Plan Vasco de FP:           |
| Tecnología, digitalización y sistemas inteligentes (págs. 65--73), al |
| integrar sensórica avanzada, recogida de datos y un modelo de         |
| inteligencia artificial para personalizar la rehabilitación física.   |
|                                                                       |
| Responde específicamente a varias líneas de actuación como:           |
|                                                                       |
| 2.1 y 2.3: desarrollo de entornos inteligentes y aplicación de IA     |
| para el seguimiento del paciente; 2.5: uso de sensórica para medir    |
| fuerza y evolución; 2.6: tratamiento y visualización de datos en la   |
| nube.                                                                 |
|                                                                       |
| Además, impulsa el trabajo interdisciplinar en FP, fomenta la         |
| innovación aplicada y refuerza la transición hacia centros 5.0, con   |
| impacto real en los ámbitos educativo y sociosanitario.               |
|                                                                       |
| El impacto de este proyecto se traduce también en su contribución a   |
| varios Objetivos de Desarrollo Sostenible. Contribuye al ODS 3 --     |
| Salud y bienestar, al mejorar la seguridad y calidad de los procesos  |
| de rehabilitación mediante datos fiables; al ODS 4 -- Educación de    |
| calidad, ofreciendo al alumnado un entorno de aprendizaje innovador y |
| contextualizado; al ODS 8 -- Trabajo decente y crecimiento económico, |
| profesionalizando el sector sociosanitario a través de la tecnología; |
| y al ODS 9 -- Industria, innovación e infraestructura, integrando     |
| soluciones disruptivas que elevan los estándares tecnológicos en el   |
| ámbito clínico.                                                       |
|                                                                       |
| El proyecto será open source y estará disponible para centros de FP,  |
| empresas sociosanitarias, entidades educativas y profesionales del    |
| sector. Su escalabilidad está garantizada gracias a su diseño modular |
| y su capacidad de adaptación a distintos entornos clínicos y          |
| educativos.                                                           |
|                                                                       |
| **ALCANCE, ESCALABILIDAD Y DESPLIEGUE**                               |
|                                                                       |
| Uno de los aspectos diferenciales de Smart Rehab Bar es su enfoque    |
| abierto, accesible y fácilmente replicable. Desde su diseño inicial,  |
| el proyecto ha sido planteado para que tanto el hardware como el      |
| software desarrollado puedan compartirse en plataformas de código     |
| abierto como GitHub. Esta decisión garantiza que los resultados       |
| obtenidos estén disponibles para otros centros de Formación           |
| Profesional, profesionales del ámbito sociosanitario o entidades      |
| interesadas, favoreciendo la reutilización, la adaptación y la mejora |
| continua del sistema.                                                 |
|                                                                       |
| El dispositivo está pensado para integrarse fácilmente en barras de   |
| sujeción ya existentes en centros de rehabilitación o educativos, sin |
| necesidad de grandes inversiones ni modificaciones estructurales.     |
| Esto permite una implantación ágil y una alta escalabilidad, tanto a  |
| nivel técnico como formativo. Su sencillez de integración, unida al   |
| potencial de análisis que ofrece la inteligencia artificial aplicada  |
| al tratamiento de los datos recogidos, lo convierte en una solución   |
| versátil, útil y adaptable a diferentes contextos.                    |
|                                                                       |
| Además, Smart Rehab Bar tiene un claro componente educativo, ya que   |
| permitirá a los estudiantes de ciclos de electrónica, informática o   |
| sanidad trabajar con un caso real, aplicando conocimientos de         |
| sensórica, programación, análisis de datos e interpretación clínica.  |
| Esta dimensión formativa facilita la transferencia del conocimiento   |
| generado durante el desarrollo del proyecto, no solo entre el         |
| profesorado y el alumnado participante, sino también hacia otros      |
| centros interesados.                                                  |
|                                                                       |
| En definitiva, se trata de una propuesta con vocación de impacto      |
| real, tanto en el ámbito clínico como en el educativo, que, apuesta   |
| por la colaboración, la tecnología útil y la mejora de la atención en |
| procesos de rehabilitación a través del uso responsable de la         |
| inteligencia artificial.                                              |
|                                                                       |
| **PLAN DE COLABORACIÓN**                                              |
|                                                                       |
| El desarrollo de Smart Rehab Bar se sustenta en una colaboración      |
| sólida entre distintos agentes educativos y profesionales, lo que     |
| aporta al proyecto una visión rica, realista y alineada con las       |
| necesidades del entorno. El liderazgo técnico recae en La Salle       |
| Berrozpe, desde donde se abordará todo el diseño electrónico del      |
| dispositivo y la integración de los sensores.                         |
|                                                                       |
| Por su parte, Nazaret se incorpora al proyecto como referente en el   |
| ámbito sociosanitario, aportando una doble dimensión clave. Por un    |
| lado, integra el dispositivo en un contexto clínico realista, desde   |
| el enfoque del acompañamiento, la seguridad del paciente y el rigor   |
| documental. Por otro lado, aporta una mirada humanista e inclusiva,   |
| asegurando que la tecnología no solo sea funcional, sino también      |
| empática, accesible y centrada en la persona. Esta aportación se      |
| materializa en propuestas pedagógicas, materiales formativos y        |
| protocolos de aplicación que priorizan la experiencia del usuario y   |
| la ética en el uso de la inteligencia artificial en salud.            |
|                                                                       |
| La principal labor de Zabalburu Ikastetxea dentro del proyecto Smart  |
| Rehab Bar se centra en el desarrollo de la aplicación de              |
| monitorización que permitirá visualizar los datos recogidos por la    |
| Smart Rehab Bar. Esta aplicación jugará un papel crucial al facilitar |
| a los profesionales de la salud una representación clara e intuitiva  |
| de la fuerza ejercida por cada extremidad del paciente durante los    |
| ejercicios de rehabilitación.                                         |
|                                                                       |
| De manera específica, Zabalburu Ikastetxea será responsable de        |
| diseñar e implementar el sistema de recogida, tratamiento y           |
| visualización de datos. Esto incluye:                                 |
|                                                                       |
| El diseño del sistema de recogida digital de datos procedentes del    |
| hardware.                                                             |
|                                                                       |
| El desarrollo del backend y/o scripts de tratamiento de datos para la |
| limpieza, clasificación y análisis básico, lo que permitirá guardar   |
| todos los datos históricos generados durante las sesiones de          |
| rehabilitación.                                                       |
|                                                                       |
| La implementación de un panel visual que permita a los profesionales  |
| consultar datos clave como la fuerza ejercida, el tiempo de uso y las |
| repeticiones realizadas. Este panel será fundamental para visualizar  |
| de manera sencilla la información recogida.                           |
|                                                                       |
| La investigación y aplicación de un modelo de IA sencillo para el     |
| análisis de la progresión del paciente, permitiendo observar la       |
| evolución a lo largo del tiempo.                                      |
|                                                                       |
| Además, se explorará la opción de analizar los históricos de datos    |
| junto con información adicional como la edad y el diagnóstico, con el |
| objetivo de mencionar la posibilidad futura de desarrollar            |
| funcionalidades básicas para la generación de posibles diagnósticos   |
| basados en datos históricos. Sin embargo, en una primera fase, el     |
| enfoque principal estará en la visualización, el almacenamiento y el  |
| análisis de la progresión del paciente.                               |
|                                                                       |
| Además, el proyecto cuenta con el respaldo y asesoramiento de         |
| fisioterapeutas del sistema público de salud (Osakidetza) y también   |
| de clínicas privadas de fisioterapia y rehabilitación, que colaboran  |
| ofreciendo una validación profesional del enfoque terapéutico, así    |
| como la identificación de necesidades reales observadas en contextos  |
| clínicos. Su implicación garantiza que la solución tecnológica        |
| desarrollada no solo sea técnicamente viable, sino también útil,      |
| aplicable y alineada con la práctica cotidiana en entornos de         |
| rehabilitación.                                                       |
|                                                                       |
| Esta cooperación entre centros formativos y profesionales de la salud |
| no solo enriquece el desarrollo del proyecto, sino que también        |
| refuerza la transferencia de conocimiento bidireccional entre el      |
| mundo educativo y el profesional, potenciando el valor formativo del  |
| proyecto para el alumnado implicado.                                  |
